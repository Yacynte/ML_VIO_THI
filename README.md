# ML_VIO_THI
ML - Aided Visual Inertial Odometry

*This project performs Machine learning monocular visual inertial odometry using images and IMU data from the KITTI dataset with calibration parameters.*

---

## Installation Guide

Follow these steps to set up and run the project in your local environment.

### 1. Clone the repository

```bash
git clone https://github.com/Yacynte/ML_VIO_THI.git
cd ML_VIO_THI
```
### 2. Create a virtual environment and install dependencies

```bash
python3 -m venv venv
source venv/bin/activate
python -m pip install --upgrade pip
pip install -r requirements.txt
```
